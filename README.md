# Udacity Blockchain Capstone

### How to run tests

* Clone [this repo](https://github.com/muradmm83/bdnd-capstone).
* `cd bdnd-capstone`.
* `npm install`.
* `cd eth-contracts`.
* `truffle compile --all`.
* Start `ganache-cli`.
* `truffle migrate --reset`.
* Run all tests `truffle test`.
